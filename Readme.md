# 操作説明
- python manageTask.py : タスク一覧を表示
- python manageTask.py [タスクID] start : タスクを開始
- python manageTask.py [タスクID] delete : タスクを削除
- python manageTask.py [タスクID] (引数) : タスクを編集※組み合わせ、順不同可
# 引数一覧
- ?[状態] :　状態を設定
- +[ {}h{}m or {}h or {}m] : 時間を加算
- +[ {}h{}m or {}h or {}m] : 時間を減算
- =[タイトル]　: タイトルを設定
- :[開始日] : 開始日を設定
- ~[終了日] : 終了日を設定
